Module Chartjs.Color

type t

When supplying colors to Chart options, you can use a number of formats. You can specify the color as a string in hexadecimal, RGB, or HSL notations. If a color is needed, but not specified, Chart.js will use the global default color. This color is stored at Chart.defaults.global.defaultColor. It is initially set to 'rgba(0, 0, 0, 0.1)'. You can also pass a CanvasGradient object. You will need to create this before passing to the chart, but using it you can achieve some interesting effects.
